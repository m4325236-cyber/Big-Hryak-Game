# 🐗 Big Hryak

**A 2D endless runner-shooter game** where you play as a brave boar! Jump, shoot, and defeat enemies while collecting power-ups.

## 🎮 Gameplay

![Gameplay](screenshots/gameplay.gif) *(Add a screenshot or GIF here later)*

## 🕹️ Controls

| Action | Key |
|--------|-----|
| Move Left | A / ← |
| Move Right | D / → |
| Jump | SPACE |
| Shoot | LMB (Left Mouse Button) |
| Restart | R |
| Fullscreen | F |
| Exit | ESC |

## ✨ Features

- **3 Enemy Types** – Weak (1 HP), Medium (2 HP), Strong (3 HP) with different colors
- **Power-ups** – ❤️ Health, ⚡ Speed boost, 💪 Double damage
- **Obstacles** – Spikes and holes to avoid
- **Adaptive Scaling** – Works on any screen resolution
- **Background Music** – Atmospheric soundtrack

📊 Scoring System
Action	Score
Hit an enemy	+5
Kill a weak enemy (1 HP)	+10
Kill a medium enemy (2 HP)	+15
Kill a strong enemy (3 HP)	+25
🎯 Game Mechanics
Enemy damage depends on their HP (stronger enemies deal more damage)

Speed power-up increases movement speed for 5 seconds

Double damage power-up makes bullets deal 2 damage for 5 seconds

Health power-up restores 1 HP

Invincibility frames after taking damage (1 second)
